# Patch-Management-Endpoint-Administration
Installation &amp; Deployment of ManageEngine Endpoint Central
Project Overview

This project focused on the installation, configuration, and deployment of ManageEngine Desktop Central (now Endpoint Central) as a centralized endpoint management solution. The implementation covered server setup, agent deployment to client machines, system visibility verification, and patch management to ensure endpoints remained secure and up to date.
